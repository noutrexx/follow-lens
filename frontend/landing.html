FAQ

Answer trust questions before a buyer hesitates.

Does it ask for my Instagram password?

No. The local version uses a browser session value. That value is sensitive, so it should stay on your machine.

Can Instagram limit the account?

Any automation can carry risk. FollowLens uses cooldowns and delays, but scans should stay conservative.

Where is the data stored?

Snapshots are stored locally in the project data folder.

Can it monitor private accounts?

Only accounts your session can already access.
